Projects and Experience
Kartus
Kartus is a chatbot hosted on the Discord messaging platform. Initially developed to simplify the complex daily schedule for students into a quickly accessible and visual format, Kartus quickly evolved to accommodate several features to solve the real problems I observed myself and the students around me facing in real time. At a time where all classes were held online, and there a palpable disconnect among the audience attending the same lectures and dealing with the same coursework, Kartus helped students with shared faculties and similar courses find each other on Discord, using the data it collected in order to visually formulate the user schedules.

The main motivation behind the project was to make online college easier, and I was able to do this by implementing solutions to the problems I faced in real-time as I went along. Kartus is a passion project which gave me a great deal of satisfaction as it was immensely helpful to my friends and other students in my batch. The project, however, was dropped around the commencement of physical classes.
Mark Registry
A Tkinter based python project which used MySQL databases to replicate an online mark registry for teachers. This application allows creation of multiple classes and addition of subjects and students in each class. Every class supports addition of multiple tests for continious assessment tests and final assessment tests. Statistics for every test can be viewed and the overall performance of the can for the test can be viewed.
